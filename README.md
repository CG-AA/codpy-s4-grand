# Codpy S4 — ESP32 作業包

兩個 ESP32 作業，一個 repo。需要一塊 ESP32 + 一條 data USB 線（非純充電線）。

> 🩺 卡關了？雙擊 [`doctor/doctor.bat`](./doctor) 做環境診斷與自動修復。

## 專案結構

```
.
├── pi_cake/   # 評分機作業：π 數字題，跑 capybara_judge.py（細節見 pi_cake/README.md）
├── fractal/   # ESP32 雙核碎形 demo，跑 render.py（細節見 fractal/README.md）
└── doctor/    # 環境診斷／修復腳本 doctor.bat（細節見 doctor/README.md）
```
